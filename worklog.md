- 重いエフェクト（魔法陣・エネルギー波・星屑）がスマホでも稼働

- 魔法陣: 15秒 → 30秒間隔（100%削減）
- エネルギー波: 8秒 → 20秒間隔（150%削減）
- 星屑: 4秒 → 10秒間隔（150%削減）

#### 3. Three.jsレンダリング最適化
**低消費電力設定**:
- `powerPreference`: "high-performance" → "low-power"（モバイル）
- `antialias`: true → false（モバイル）
- `devicePixelRatio`: 最大2 → 1（モバイル）

#### 4. 重いエフェクトのモバイル無効化
**完全無効化対象**:
- 魔法陣エフェクト
- エネルギー波エフェクト
